# Word Indexing System (AVL & Hash Table) 

This project is a C application that indexes words from a text file using **AVL Tree** and **Hash Table**.

The system reads words from a file, stores them in an AVL tree to maintain sorted order, and tracks their frequency. Then, the data is transferred to a hash table for fast searching and retrieval.

It allows users to insert, delete, and search words, as well as display word frequencies and statistics such as most frequent words and total unique words.

This project demonstrates the use of **AVL Trees and Hash Tables** for efficient data storage, searching, and processing.
